# FlutterMobil
Bu proje, Flutter'da **Provider, Riverpod ve BLoC** kullanarak durum yönetimi (state management) yaklaşımlarını karşılaştırmayı amaçlayan bir örnek uygulamadır. Projede basit bir sayaç uygulaması yapılmıştır.

---

# Proje Amacı
- Flutter'da farklı state management kütüphanelerini öğrenmek ve karşılaştırmak.
- Provider, Riverpod ve BLoC'un performans, kod karmaşıklığı, öğrenme eğrisi ve ölçeklenebilirlik açısından farklarını görmek.

---

# Kullanım
- Uygulama açıldığında ekranda 0 sayısı görünür. 
- `+` butonuna tıkladığınızda sayı artar. 
- Bu örnek, Provider, Riverpod ve BLoC ile aynı işlevi gerçekleştiren farklı state management yaklaşımlarını göstermektedir.

---

# Kullanılan Paketler
- provider
- flutter_riverpod
- flutter_bloc

---

# Kurulum

1. Flutter SDK' yı kurun 
2. Projeyi klonlayın veya indirin.
3. Terminal veya VS Code'da proje klasörüne gidin:

```bash
cd fluttermobil


```

4. Gerekli paketleri yükleyin
```bash
flutter pub get
```

5. Uygulamayı çalıştırın
```bash
flutter run
```
-**Kübra Celep**
